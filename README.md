「かささす」Androidアプリ
====
～うっかり傘を持っていくのを忘れてしまうあなたに～


## Description
うっかり傘を持っていくのを忘れてしまうことを防ぐアプリです．　　
玄関から出る直前のタイミングで「傘持った？？」と聞いてくれます．
